Introduction

Angular est un framework JavaScript qui permet de développer des SPA (Single Page Application).

Définitions :

SPA : application qui tournent dans une seule page HTML grâce au Javascript.

Framework : ensemble d'outils et de composants à partir desquels on peut développer des applications.

Angular passe par 3 langages principaux :

  • HTML
  • SCSS
  • TypeScript : permet un typage strict

Les avantages d'Angular :

  • Framework complet (pas besoin de librairie supplémentaire
  • Permet de mettre en place les bonnes pratiques
  • Conçu pour fonctionner avec TypeScript

Les dépendances

  • Node
  • Npm
Définitions :

LTS (Long Term Support) : une version stable spécifique qui bénéficiera de support pendant plus longtemps qu'une version standard.

Installer le CLI

CLI : Command Line Interface

⚠️ Sous MAC : possible erreur de gestion de droits. DOC

  
    // install le CLI de manière générale
    npm i -g @angular/cli

    // version
    ng v

    => ng correspond au CLI d'Angular

Nouvelle application

  
    // install le projet dans le  dossier dans lequel on se trouve
    ng new mon-projet --directory ./

    // install le projet et crée un nouveau  dossier avec le nom du projet
    ng new mon-projet  --style=scss --skip-tests=true

     flag : --style

    // lancer le serveur de développement
    ng serve --o

    // l'application s'ouvre à l'adresse : 
    localhost:4200

     --o permet d'ouvrir l'application dans le navigateur
