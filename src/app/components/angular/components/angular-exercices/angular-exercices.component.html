Mise en pratique : clone instagramme

  1. Mettre en place le projet
  2. Créer un component "post"
  3. Afficher des données fictives venant du TS
  4. Créer un bouton like et le lier à une méthode TS (au click = + 1 like, nouveau click = - 1 like)
  5. Créer une class dans un dossier "model" qui contient le constructeur d'un post
  6. Ajouter des propriétés personnalisées au component
