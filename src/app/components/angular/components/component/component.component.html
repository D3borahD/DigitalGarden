Component

Les composants sont la base d'une application Angular.

Ce sont des balises HTML personnalisées

Une application peut-être vue comme une arborescence de composants, avec AppComponent comme composant racine.

Un composant est réutilisable.

Créer un composant

  Bash
  
    ng generate component mon-composant

    ng g c mon-composant

Un composant contient 3 ou 4 fichiers :

  • HTML => Template
  • TS
  • CSS/SCSS/...
  • spec : fichier de test
  TypeScript
  
    import { Component } from '@angular/core';

    @Component({
      selector: 'app-component',
      templateUrl: './component.component.html',
      styleUrls: ['./component.component.scss']
    })

    export class AngularIntroductionComponent {
      title = 'Mon post'
    }

    // HTML
    {{ title }}

La classe est déclarée avec un décorateur @Component

Un @Décorateur permet d'apporter des modifications à une classe.

title = 'Mon post' est un attribut de la class AppComponant.

Lorsque l'on déclare un attribut de classe il n'est pas précédé de let, const, ...

Pour être utilisé, un composant doit être déclaré dans app.module.ts

app-module.ts est le module principal de l'application.

  TypeScript
  
    import { NgModule } from '@angular/core';
    import { BrowserModule } from '@angular/platform-browser';

    import { AppComponent } from './app.component';
    import { PostComponent } from './post/post.component';

    @NgModule({
      declarations: [
        AppComponent,
        MonComponent,
      ],
      imports: [
        BrowserModule
      ],
      providers: [],
      bootstrap: [AppComponent]
    })
    export class AppModule { }
