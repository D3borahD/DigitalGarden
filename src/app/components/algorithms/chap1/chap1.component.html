Chapitre 1 : Le rôle de l'algorithmiques dans l'informatique

Une instance d'un problème est l'entrée nécessaire (satisfaisant toutes les contraintes imposées dans l'énoncé du problème) pour calculer une solution au problème.

Parce que beaucoup de programme utilisent des étapes intermédiaires, le tri est une opération fondamentale en informatique.

Donc il y a un grand nombre d'algorithmes de tri.

Le meilleur ? Cela dépend de beaucoup de facteurs :

  • Le nombre d'items à trier ;
  • La mesure dans laquelle les éléments dont déjà un peu trié ;
  • Les éventuelles restrictions sur les valeurs des éléments ;
  • L'architecture de l'ordinateur et le périphérique de stockage.
